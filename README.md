# Bitcoin-USD Chart
Chart example for bitcoin - usd for last 100 days

## Getting Started
Run the development server:

```bash
chmod +x run.sh
./run.sh
```

Open [http://localhost:5000](http://localhost:5000) with your browser to see the result.
